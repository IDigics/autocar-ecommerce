# 🚗 Boutique AutoCar'z

Bienvenue dans le projet **Boutique AutoCar'z**, une section e-commerce dédiée à la vente de voitures de luxe dans le cadre du site principal **AutoCar'z**.

> Conduisez l'exception, Ressentez la passion... Vivez AutoCar'z.

---

## 🎯 Objectif

Créer une interface boutique élégante, performante et responsive qui permet aux utilisateurs de :

- Explorer une collection de voitures de luxe
- Filtrer les produits
- Accéder aux détails d’un véhicule
- (À venir) Ajouter au panier

---

## 🛠️ Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **Tailwind CSS** (UI minimaliste et luxueux)
- **TypeScript**
- **Axios** (pour les appels API futurs)
- **Git / GitHub** (travail collaboratif)

---

## 🚀 Lancer le projet localement

```bash
# Cloner le repo
git clone https://github.com/IDigics/autocar-ecommerce.git

# Aller dans le dossier boutique
cd boutique-autocarz

# Installer les dépendances
npm install

# Démarrer le projet
npm run dev
```
